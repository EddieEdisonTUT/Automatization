HTML Tutorial

What is HTML?

HTML stands for HyperText Markup Language. It is the standard markup language for creating web pages and web applications.

HTML is used to structure the content on a web page and is made up of various elements.

Here's an example of an HTML element: <p>This is a paragraph</p>

HTML Elements

HTML elements are represented by tags. Tags usually come in pairs: an opening tag and a closing tag.

The opening tag indicates the start of an element, while the closing tag indicates the end. For example: <h1>This is a heading</h1>

Some HTML elements, like <br> and <img>, are self-closing and do not require a closing tag.
